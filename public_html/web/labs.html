LAB 1

GOODNIGHT MOON

For this lab, the assignment was to hand code 34 html pages each with a single line of the poem "Goodnight Moon" by Margaret Wise Brown. Each page includes a link to go forward one line and move backwards one line, so the user can navigate through the poem.

LAB 2

WIKI

For this lab, the assignment was to take a wikipedia page and copy all of the text to a hand coded html document that was properly structured using HTML5 tags. The page needed to include an internal navigation, as well as a few absolute links. I used the random article generator to find a reasonably sized article about a synagogue.
