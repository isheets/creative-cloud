HYPERTEXT NARRATIVE

The hypertext narrative project is a way to break up an essay into a non-linear set of web pages, which gives the user control over their viewpoint and understanding. For my project, the first link on each page progresses linearly through the essay, whereas the other two links are to random pages. Enjoy!

CSS ZEN GARDEN

For the zen garden project we followed the official guidelines for CSS Zen Garden, thus we could only edit the css of the page to stylize the html and make something that was layed out well and looked nice. Additionally, this project also introduces responsive layout — it maintains good design regardless of the viewer's device or screen size.
